# stretch-discord

A discord add-on that can run various bots:

-   A bot that remind us to stretch every hour
-   A bot to play sudoku

# For development

## Install

`make install`

## Run

`make start`

## Get logs

`make logs`

# For production

(Need ssh acces to discord-bot server)

## Deployment

`make deploy`

## Run

`make start-server`

## Stop

`make stop-server`

## Get logs

`make logs-server`
